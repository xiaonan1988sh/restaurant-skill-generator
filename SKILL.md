# 🍜 餐厅 AI 名片生成器

帮助餐饮商家通过对话生成自己的 AI 名片（AI Skill），无需懂代码。

## 工作流程

### 第一步：收集信息

通过自然对话向商家收集以下信息。**必填项**标 ⭐，其余可跳过。

| 字段 | 说明 | 示例 |
|------|------|------|
| ⭐ 餐厅名称 | 中文店名 | 金谷园饺子馆 |
| ⭐ 英文标识 | 用于文件夹名，纯英文小写+连字符 | jinguyuan |
| ⭐ 一句话介绍 | 15字以内 | 手工现包饺子，30年老味道 |
| ⭐ 主营品类 | 菜系/品类 | 饺子 |
| ⭐ 营业时间 | | 10:00-21:00 |
| ⭐ 门店1名称+地址 | 至少一家 | 总店 / 北京市朝阳区XX路1号 |
| 门店2名称+地址 | 可选 | |
| WiFi名称/密码 | 可选 | |
| 支持外卖 | 是/否 | 否 |
| 支持打包 | 是/否 | 是 |
| 品牌关键词 | 逗号分隔 | 必吃榜,老字号 |
| 说话风格 | AI回答时的语气 | 热情朴实 |
| ⭐ 作者名称 | 发布者署名 | 张老板 |

**对话引导技巧：**
- 先问店名，自然延伸到品类、地址
- 别一次问完所有字段，分 2-3 轮
- 用已有信息推测缺失项（如从店名猜品类），让商家确认即可
- 可选项统一问"还有什么要补充的？WiFi、外卖之类的？"

### 第二步：生成配置文件

收集完信息后，将内容写入 `{baseDir}/我的餐厅信息.txt`，格式：

```
餐厅名称=金谷园饺子馆
英文标识=jinguyuan
一句话介绍=手工现包饺子30年老味道
主营品类=饺子
营业时间=10:00-21:00
门店1名称=总店
门店1地址=北京市朝阳区XX路1号
门店2名称=
门店2地址=
WiFi名称=
WiFi密码=
支持外卖=否
支持打包=是
品牌关键词=必吃榜,老字号
说话风格=热情朴实
作者名称=张老板
```

### 第三步：运行生成器

```bash
cd {baseDir} && python3 generate.py 我的餐厅信息.txt
```

生成结果在 `{baseDir}/<英文标识>-skill/` 目录下。

### 第四步：引导发布

生成成功后，告诉商家：

1. 生成的文件在 `<英文标识>-skill/` 文件夹里
2. 打开里面的《如何发布你的AI名片.md》有保姆级教程
3. 核心步骤：注册 Gitee/GitHub → 建仓库 → 上传文件
4. 发布后把链接分享给顾客，顾客告诉自己的 AI 助手就能安装

如果商家装了 `clawhub` CLI，也可以直接：
```bash
cd <英文标识>-skill && clawhub publish . --slug <英文标识>-skill --name "XX餐厅AI名片" --version 1.0.0
```

## 注意事项

- 英文标识只允许小写字母、数字、连字符，不能有空格或中文
- 如果商家给的信息不完整，用已有信息先生成，后续可以再改
- 生成器可以反复运行，新结果会覆盖旧文件
